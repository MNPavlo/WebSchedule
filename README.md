# WebSchedule
1. В даній роботі було спроектовано та розроблено програмну систему з багаторівневою системною архітектурою, яка виконує функції системи електронного розкладу. Було реалізовано можливість реєстрації користувачів, авторизацію. Користувачам надається певна роль завдяки якій він має доступ лише до деяких функцій.
2. Для проектування і розробки даної системи було використано IDE Microsoft Visual Studio Community 2019. Для створення діаграм було використано програмний засіб Microsoft Visio 2019. Для контролю версій використовувалася система GitHub.
3. Було узагальнено та використано свої знання з об’єктно-орієнтованого програмування на мові С# при використанні методів проектування та розробки багаторівневої архітектури та побудовано діаграму класів.
4. Було реалізовано багаторівневу архітектуру програми та вивчено правила «високого» стилю програмування та застосовано їх під час написання програми.
5. В ході виконання курсової роботи було удосконалено навички роботи у команді, які є одними з ключових у роботі розробника програмного забезпечення, також навички розподілення ролей та обов’язків між членами команди.
